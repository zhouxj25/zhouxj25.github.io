Vmware中Ubuntu安装tldk

Posted on 2018-10-27 | In blog

1. 准备

tldk简介：

  1. 做为第四层协议（TCP、UDP等）库的集合。
  2. 做为VPP图形节点以及插件等，使用这些库实现主机栈（host stack）。
  3. 创建这样的机制（netlink代理，打包等），使现有不支持VPP的软件轻松的使用产生的主机栈。
    系统版本

dpdk版本

dpdk-18.05.1

tldk版本

master

网卡类型

2. 编译DPDK

按照《Vmware中Ubuntu安装DPDK》先安装DPDK。

3. 下载源码

git clone https://gerrit.fd.io/r/tldk

4. 编译代码

4.1. 编译tldk库文件

如果想调试库文件代码，设置以下环境变量：

export EXTRA_CFLAGS=”-g3 -O0”

然后make进行编译，编译完成之后在/etc/profile文件中设置以下环境变量：

export TLDK_ROOT=/root/fdio/tldk
export C_INCLUDE_PATH=$C_INCLUDE_PATH:$TLDK_ROOT/$RTE_TARGET/include
export LIBRARY_PATH=$LIBRARY_PATH:$TLDK_ROOT/$RTE_TARGET/lib

执行以下命令，使环境变量生效。

4.2. 编译l4fwd样例

关于l4fwd的详细使用说明，请参考文档《tldk l4fwd用例说明》。
